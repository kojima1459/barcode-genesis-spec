## 🎮 プロジェクトについて

バーコード・ジェネシスは、身の回りにあるバーコードをスキャンすることで、AIが自動生成したユニークなロボットを入手し、他のプレイヤーと対戦できるゲームアプリです。

**コンセプト**: "日常に潜むモンスターを見つけ出し、最強のバトラーを目指せ！"

## 🛠️ 技術スタック

- **フロントエンド**: Flutter
- **バックエンド**: Go (Golang)
- **データベース**: PostgreSQL
- **インフラ**: Google Cloud (GCP)
- **AI生成ロジック**: Python (Cloud Functions)
